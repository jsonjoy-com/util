Class Codegen<Fn, Linkable>

A helper class which helps with building JavaScript code for a single function. It keeps track of external dependencies, internally generated constants, and execution steps, which at the end are all converted to to an executable JavaScript function.

The final output is a JavaScript function enclosed in a closure:

(function(d1, d2, d3) {
  var c1 = something;
  var c2 = something;
  var c3 = something;
  return function(r0) {
    var r1 = something;
    var r2 = something;
    var r3 = something;
    return something;
  }
})

Where d* are the external dependencies, c* are the internal constants, and r* are the local immutable infinite registers.

getRegister

  • getRegister(): string

  • Codegen uses the idea of infinite registers. It starts with 0 and increments it by one for each new register. Best practice is to use a new register for each new variable and keep them immutable.

    Usage:

    const r = codegen.getRegister();
    codegen.js(`const ${r} = 1;`);

    Returns string

    a unique identifier for a variable.
